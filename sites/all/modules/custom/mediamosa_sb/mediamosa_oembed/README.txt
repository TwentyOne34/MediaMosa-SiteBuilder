
MediaMosa oEmbed module

Makes all assets available to the oEmbed provider.

Installation
------------

This module depends on the oembed module (drupal.org/project/oembed).
Other than that just enable the module. 

Usage
-----

Go to
http://<drupalinstall>/oembed/endpoint?url=http://<drupalinstall>/asset/<id>
to obtain an oEmbed code for an asset.
